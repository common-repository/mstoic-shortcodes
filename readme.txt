Adds support for many shortcodes to your WordPress website.

== Description ==

This plugins add support for many shortcodes. It also adds a shortcode generator button to the post editor screen which makes adding shortcodes a breeze. Live preview can be seen before even placing the shortcodes.

See the live preview of all the [shortcodes](http://www.mstoicthemes.com/shortcodes-list/).

= List of Available Shortcodes =
* Clear: Clears the floating elements.
* Buttons: Add buttons in your content. We also have two amazing effects for buttons.
* Expendable Box: A box that display more content with a click of a button.
* Facebook Like Box: Add a Facebook like box. Show/Hide Faces. Show/Hide Share Button.
* Google Maps: Embed Google Maps anywhere you want. Just type the name of the place, and you have a fully functional Google Map.
* Images: Add images to your content in a beautiful layout. Optionally give Heading and Sub-heading. You also have the option to round the images.
* Slider Images: Add slider in your post/page content.
* Space: Clears the floating elements and adds some spacing.
* Testimonials: Display testimonials in a beautiful new way.
* Twitter Follow Button: Add a Twitter follow button. Choose from small/large button. Show/Hide Follower count. Show/Hide Twitter username.
* YouTube Embeds: Embed YouTube videos with a lot of customization options.

== Installation ==

Upload, Activate, and Use.
